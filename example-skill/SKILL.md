# Example Skill

This is an example skill that demonstrates the basic structure and format for creating custom skills.

## Purpose

This skill serves as a template for creating new skills. It shows the required YAML frontmatter and provides guidance on how to structure skill instructions.

## Usage

When you want to use this skill, simply mention it by name:

"Use the example-skill to help me with..."

## Skill Instructions

This section contains the actual instructions that Claude will follow when this skill is invoked.

### Step 1: Understand the Request

First, carefully read and understand what the user is asking for.

### Step 2: Apply Specialized Knowledge

Use the specialized knowledge and patterns defined in this skill to address the user's request.

### Step 3: Provide Clear Output

Deliver clear, actionable results that directly address the user's needs.

## Best Practices

- Keep instructions clear and concise
- Use examples where helpful
- Define any specific tools or approaches that should be used
- Include error handling guidance if applicable

## Extending This Template

To create your own skill:

1. Copy this directory
2. Rename it to match your skill's purpose
3. Update the YAML frontmatter (name, description, license)
4. Replace this content with your skill-specific instructions
5. Add any supporting files (scripts, templates, etc.) as needed
